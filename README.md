Hello GitHub: erstes Repository online angelegt.

Spiele Erklärung: Man sucht sich eine Zahl zwischen 1 und 100 aus und das spiel versucht diese zu erraten mithilfe einer binären Suche 
Higher Button: Nimmt die Zahl die gerade angezeigt wird und macht sie zum neuen Minimum
Lower Button: Nimmt die Zahl die gerade angezeigt wird und macht sie zum neuen Maximum
Correct Button: Zeigt einen Text für das Ende des Spiels an und deaktiviert alle Buttons außer den Restart Button
Restart Button: Startet das Spiel mit den zurückgesetzten minimum und maximum Werten neu
Binäre Suche: Ein Algorythmus der versucht einen Wert zu finden indem er den Anfangswert immer mehr halbiert
Unity Version: Unity 6.0.57f1
Startszene: MainScene
